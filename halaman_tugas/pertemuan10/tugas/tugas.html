CSS ( CASCAding Style Sheets )
Keuntungan CSS :
1. Menentukan tampilan dari halaman-halaman web dalam sebuah tempat
tanpa mengulangnya kembali melalui script HTML anda.
2. Menentukan isi hingga ukuran piksel dari tiap halaman web dengan akurat
Kelemahan CSS :
CSS hanya dapat bekerja dalam browser versi 4.
Namun dewasa ini lebih dari 95% dari keseluruhan browser telah banyak yang mendukungnya
